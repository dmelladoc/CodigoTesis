# Notebooks de desarrollo

Los siguientes notebooks son los utilizados dentro del proceso de entrenamiento y pruebas iniciales.
Estos se encuentran en diferente grado de documentación o completitud, y levemente anonimizados debido a sus estructuras.

Estos no son directamente compatibles con el codigo final, hay que hacer algunas modificaciones dentro de las importaciones.
Estos se mantienen aqui como documentación de algunos de los procesos realizados.
Se espera a futuro re-implementar la etapa de entrenamiento, adaptada para este proceso

| Archivo | Descripcion |
|:--------:|:-----------|
|[`01-ModelSelection`](01-ModelSelection.ipynb) | Construcción de dataset, prueba incompleta de modelos |
|[`02-GridSearch`](02-GridSearch.ipynb)| Busqueda de hiperparametros, requiere raytune |
|[`03-ClassifierTrain`](03-ClassifierTrain.ipynb) | Entrenamiento de modelo de detección de hallazgos |
|[`04-ClassifierMetrics`](04-ClassifierMetrics.ipynb) | Metricas de evaluacion de modelo |
|[`06C-PointingGameEval`](06C-PointingGameEval.ipynb) | Evaluacion de conjunto de test utilizando pointing game |
|[`Corrmedia`](Corrmedia.ipynb) | Evaluación de precision vs numero de mascaras |
